This program runs and creates and displays a binary tree as well
as the traversal methods. The program also creats output files
that log every tree's preorder, postorder and inorder traversal
results.

The user can choose to either make a tree or quit the program. 
If the user wants to create a tree but has an invalid tree or
filename they will be notified imediatly and will be given the
option to try again or quit the program.
